{# Browser-side sync: fizzy's Activity Tracker integration #}

Browser-Side Sync

Use when the server's IP is blocked by JCink. Add this snippet to your Activity Tracker page to send ACP data to the Watcher after each dump: 

<script>
document.getElementById('build-table').addEventListener('click', function() {
  setTimeout(function() {
    if (typeof rawArray !== 'undefined' && rawArray.posts) {
      fetch('{{ request.base_url }}api/acp/upload-dump', {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify(rawArray)
      }).then(r => r.json()).then(d => console.log('Watcher sync:', d));
    }
  }, 2000);
});
</script>
