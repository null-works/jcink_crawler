{# Browser Sync Setup Instructions (hidden by default) #}

Browser Sync Setup

One-time setup: create a board page on your JCink forum that the Watcher uses as a bridge.

  1. Go to your JCink ACP → Skins & Templates → Board Page HTML
  2. Click Create New Page
  3. Set the page key to: watcher-sync
  4. Paste this into the HTML content:
<div id="watcher-sync" style="padding:20px;font-family:monospace;color:#888;">
  Watcher Browser Sync bridge. Do not close this page.
</div>
<script src="{{ request.base_url }}static/js/browser_sync.js"></script>
  1. Save the page
  2. Verify it loads: {{ forum_base_url }}/index.php?act=boardpage&page=watcher-sync
  3. Done! The Browser Sync button above will now work.
